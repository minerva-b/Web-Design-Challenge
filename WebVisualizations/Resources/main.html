Project Overview:

WeatherPy

The purpose of this project is to analyze how the weather changes as you move closer or farther away from the Equator. First, we must use the OpenWeatherMap API to call and pull data on 500+ cities around the world. The goal is to compile a diverse set of coordinates (latidude, longitude) to observe from places around the world.
Once we have compiled a list of at least 500 cities we use tools such as Matplotlib (a python library) to plot the various factors of the weather. These factors include the max temperature, humidity, cloudiness, and wind speed up against the latitude of the cities in our dataset. See 'Data' to view the compiled 500+ cities for this project. You can also find the data visualizations on this site along with explanations of the trends and correlations.
